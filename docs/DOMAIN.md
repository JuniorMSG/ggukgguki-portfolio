# 꾹꾹이 도메인 지식

## 자산 구조

```
자본 총액 (totalCapital)
├── 투자 자산 (totalInvestment)
│   ├── 순 투자자산 (= totalInvestment - acctCash)
│   │   ├── 해외계좌 (acctOverseas) — USD 기반
│   │   ├── 국내계좌 (acctDomestic) — KRW
│   │   ├── IRP (acctIrp)
│   │   ├── 연금저축-1 (acctPension1)
│   │   ├── 연금저축-2 (acctPension2)
│   │   └── ISA (acctIsa)
│   └── 유동현금 (acctCash)
│       ├── 청년도약계좌
│       ├── 발행어음 (단기/장기)
│       ├── CMA (1, 2)
│       ├── 달러
│       └── 기타 현금
└── 비유동현금 (= totalCapital - totalInvestment)
    ├── 부모님
    └── 청약
```

## 용어 정의

| 용어 | 계산 | 설명 |
|------|------|------|
| **자본 총액** | 스냅샷 totalCapital | 모든 자산의 합계 (투자 + 유동현금 + 비유동현금) |
| **투자 자산** | 스냅샷 totalInvestment | 주식 + 채권 + 유동현금 (비유동 제외) |
| **순 투자자산** | totalInvestment - acctCash | 주식 + 채권만 (현금 완전 제외) |
| **유동현금** | acctCash | 즉시 투자 가능한 현금 (CMA, 발행어음, 도약계좌 등) |
| **비유동현금** | totalCapital - totalInvestment | 당장 투자에 쓸 수 없는 현금 (부모님, 청약 등) |
| **누적 투자금** | DCA 전체 합산 | 지금까지 투자 계좌에 넣은 원금 총합 |
| **수익률** | (투자자산 - 투자원금) / 투자원금 * 100 | 투자 원금 대비 자산 증가율 |
| **순증가액** | 이번주 자본 - 지난주 자본 - 추가입금 | 순수 시장 변동으로 인한 자산 변화 |
| **매수금액** | 수량 × 매수가 | 종목 매입에 사용한 금액 (평가금액 아님) |
| **평가금액** | 수량 × 현재가 | 현재 시장가 기준 종목 가치 (미구현) |

## 주간 스냅샷

매주 일요일 기준으로 포트폴리오 현황을 기록.

### 저장하는 기초 데이터
- 주차, 시작일, 종료일
- 자본 총액, 투자 자산 총액
- 계좌별 평가금액 (해외, 국내, IRP, 연금1, 연금2, ISA, 현금)
- 자산군별 (성장주, 배당주, 채권, 현금, 국장)
- 배당금 (주간/누적)
- 환율 (USD/KRW)

### 계산으로 도출하는 데이터
- 성장률 = 이번주 / 지난주
- 수익률 = (투자자산 - 투자원금) / 투자원금
- 순증가액 = 이번주 - 지난주 (weeklyChange)
- 자산군 비율 = 각 자산군 / 전체

## 연봉 계산기

- **간이세액표**: 국세청 발표, 646개 구간, 월급여 × 부양가족수 → 소득세
- **비과세**: 기본 월 20만원 (식대), 4대보험/세금 계산에서 제외
- **4대보험**: 국민연금 4.75%, 건보 3.595%, 장기요양 건보의 13.14%, 고용 0.9% (2026년)
- **이자소득세**: 15.4% (현금성 자산 이자에 적용)

## 투자 전략 (MSG 기준)

- **DCA (Dollar Cost Averaging)**: 매주 정액 투자 (주 100만원 목표)
- **자산배분**: 성장주 50% / 배당성장주 40% / 현금 10%
- **세제 활용**: 연금저축 + IRP 세액공제 최대화, ISA 비과세 한도 활용
