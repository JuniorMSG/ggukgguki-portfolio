# DCA 도메인

## 개요

DCA (Dollar Cost Averaging) — 정액 적립식 투자. 매주 일정 금액을 계좌에 입금하여 시장 타이밍 리스크를 분산.

## 데이터 구조

| 필드 | 설명 |
|------|------|
| account_id | 입금 계좌 |
| amount | 투자 금액 (원). 음수 = 출금 |
| record_date | 투자일 |
| memo | 주차 라벨 등 메모 (24-01 주차) |

## 한도 관리

- 각 계좌는 **연도별 한도** (account_annual_limit)를 가짐
- 올해 DCA 합산 vs 한도로 **소진율** 계산
- 소진율 = 올해 투자금 / 올해 한도 × 100

## 연도별 요약

DCA 페이지에서 연도 × 계좌 매트릭스로 표시:

```
       연금저축1  연금저축2  IRP    ISA       해외계좌  일반-국내
2024   600만     900만    300만  3,400만   -        -
2025   600만     900만    300만  2,203만   -        -
2026   601만     -        -      101만     400만    -
합계   ...
```

## 누적 투자금

```
누적 투자금 = 전체 DCA 레코드의 amount 합산 (2019~현재)
```

대시보드 총자산 탭에 표시.
