# 수입/지출 도메인

## 카테고리 체계 (2단 구조)

### 수입 (INCOME)
```
수입 (100)
├── 급여 (101)
├── 부수입 (102)
├── 상여금 (103)
└── 은행이자 (104)
```

### 지출 — 고정비 (EXPENSE)
```
고정비 (200)
├── 보험 (201)
├── 구독료 (202)
├── 통신비 (203)
├── 교통비 (204)
├── 주거비 (205)
├── 도시가스 (206)
├── 전기 (207)
├── 관리비 (208)
├── 기타고정비 (209)
└── 구독/패스 (210)
```

### 지출 — 생활비 (EXPENSE)
```
생활비 (300)
├── 식비 (301)
├── 쇼핑 (302)
├── 여가 (303)
├── 커피 (304)
└── 기타비용 (305)
```

### 지출 — 비상금 (EXPENSE)
```
비상금 (400)
├── 의료비 (401)
└── 경조사 (402)
```

## 지출 분류 표시 순서

고정비 → 생활비 → 비상금 (데이터 없어도 항상 표시)

## 조회 기간

- **월별**: 선택한 연도/월의 기록
- **전체**: 선택한 연도의 1~12월 전체
- 연도 드롭다운으로 과거 데이터 조회 가능

## 저축률

```
투자 가능 = 수입 - 지출
저축률 = 투자 가능 / 수입 × 100
```
