# 계산기 도메인

## 연봉 실수령액 계산기

### 간이세액표
- 국세청 발표, **646개 구간** (월급여 77만~1,000만+)
- 부양가족 1~11명별 소득세 조회
- `frontend/src/data/simpleTaxTable2026.json`에 원본 저장
- 연도별 JSON 파일로 관리 (2026년만 현재 있음)

### 계산 흐름
```
1. 월급여 = 연봉 / 12
2. 과세 월급 = 월급여 - 비과세(월)
3. 4대보험 = 과세 월급 기준으로 각 요율 적용
4. 소득세 = 간이세액표에서 과세 월급 × 부양가족수로 조회
5. 자녀세액공제 차감 (8~20세: 1명 12,500원, 2명 29,160원)
6. 지방소득세 = 소득세 × 10%
7. 실수령 = 월급여 - 4대보험 - 소득세 - 지방소득세
```

- 기본 월 20만원 (식대)
- 비과세 금액은 4대보험 + 세금 계산에서 제외

## 세금 시뮬레이터

계좌 유형별 세금 비교:
- **연금저축**: 과세이연 → 수령 시 3.3~5.5% (연금소득세)
- **ISA**: 200만 비과세 + 초과분 9.9% 분리과세
- **일반계좌**: 배당 15.4%, 양도세 22% (250만 비과세)
- **해외계좌**: 양도세 22% (250만 비과세)
