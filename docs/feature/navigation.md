# 네비게이션 구조 정의

## 구조

```
[게시판 ▾]  [대시보드]  [자산관리 ▾]  [계산기 ▾]
  ├── 공지사항          ├── 자산        ├── 세금 시뮬
  └── 요청사항          ├── 자산배분     ├── 연봉
                       ├── DCA        └── 프리랜서
                       └── 수입/지출
```

## 라우팅

| 메뉴 | 그룹 | 경로 | 페이지 |
|------|------|------|--------|
| 공지사항 | 게시판 | /notices | NoticesPage |
| 요청사항 | 게시판 | /requests | RequestsPage |
| 대시보드 | - | / | DashboardPage |
| 자산 | 자산관리 | /assets | AccountsPage |
| 자산배분 | 자산관리 | /allocation | AssetsPage |
| DCA | 자산관리 | /dca | DcaPage |
| 수입/지출 | 자산관리 | /cashflow | CashflowPage |
| 세금 시뮬 | 계산기 | /tax | TaxSimPage |
| 연봉 | 계산기 | /salary | SalaryCalcPage |
| 프리랜서 | 계산기 | /freelancer | FreelancerCalcPage |

## 드롭다운 동작

- **게시판 ▾**: 클릭하면 드롭다운 (공지사항 / 요청사항)
- **대시보드**: 바로 이동 (드롭다운 없음)
- **자산관리 ▾**: 클릭하면 드롭다운 (자산 / 자산배분 / DCA / 수입지출)
- **계산기 ▾**: 클릭하면 드롭다운 (세금 시뮬 / 연봉 / 프리랜서)

## 대시보드 탭

대시보드 내부에서 탭으로 전환:

| 탭 | 설명 |
|----|------|
| 총자산 | 자본총액, 순투자자산, 현금, 수익률, 배당, 연도별 순증가, 계좌별 현황 |
| 투자현황 | 올해 투자, 한도 소진, 계좌별/월별 투자, 연도별 누적 |
| 주간추이 | 스냅샷 기반 주간 테이블, 계좌별 변화 |
| 수입/지출 | 이번달 요약, 연간요약, 지출 분류, 월별 추이 |
