# 네비게이션 — 배경 & 설계

## 구조

```
[게시판 ▾]  [자산관리 ▾]              [계산기 ▾]
  ├── 공지사항  ├── 대시보드            ├── 세금 시뮬
  └── 요청사항  ├── 자산               ├── 연봉
               ├── 자산배분            └── 프리랜서
               ├── DCA
               └── 수입/지출
```

## 공개/인증 구분

| 영역 | 로그인 필요 | 비고 |
|------|------------|------|
| 랜딩 (`/`) | X | 기능 소개 + 시작하기 |
| 계산기 | X | 세금/연봉/프리랜서 |
| 게시판 | O | 공지사항/요청사항 |
| 자산관리 | O | 대시보드/자산/자산배분/DCA/수입지출 |
| 프로필 | O | 닉네임 변경, 로그아웃 |

## 드롭다운 동작

- **게시판 ▾**: 클릭하면 드롭다운 (공지사항 / 요청사항) — 로그인 시에만 표시
- **자산관리 ▾**: 클릭하면 드롭다운 (대시보드 / 자산 / 자산배분 / DCA / 수입지출) — 로그인 시에만 표시
- **계산기 ▾**: 클릭하면 드롭다운 (세금 시뮬 / 연봉 / 프리랜서) — 항상 표시
- 비로그인 시 오른쪽에 "로그인" 링크, 로그인 시 닉네임 + 로그아웃

## 대시보드 탭

| 탭 | 설명 |
|----|------|
| 총자산 | 자본총액, 순투자자산, 현금, 수익률, 배당, 연도별 순증가, 계좌별 현황 |
| 투자현황 | 올해 투자, 한도 소진, 계좌별/월별 투자, 연도별 누적 |
| 주간추이 | 스냅샷 기반 주간 테이블, 계좌별 변화 |
| 수입/지출 | 이번달 요약, 연간요약, 지출 분류, 월별 추이 |
